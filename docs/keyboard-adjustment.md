# 键盘调整功能

Kime 输入法提供了灵活的键盘调整功能，让用户可以根据自己的使用习惯和设备特点自定义键盘的外观和行为。

## 功能特性

### 1. 键盘高度调整
Kime 支持键盘高度的动态调整，用户可以通过拖拽来改变键盘的高度：

- **拖拽调整**：在键盘界面上会出现调整控件，用户可以拖动来实时预览键盘高度
- **默认高度**：键盘有默认的高度设置，可以一键恢复
- **高度范围**：键盘高度限制在屏幕高度的 1/3 到 1/2 之间，以确保可用性
- **确认保存**：调整完成后点击确认按钮，新的高度设置会被保存并应用

### 2. 统一的键盘尺寸配置
Kime 使用 [KeyboardDimensions](file:///c:/Users/kingz/Documents/GitHub/Kime/app/src/main/java/com/kingzcheung/kime/ui/KeyboardDimensions.kt#L6-L29) 对象统一管理键盘的各种尺寸参数：

- **按键尺寸**：按键高度为 44dp，圆角为 4dp
- **气泡尺寸**：上滑气泡大小为 48dp，下滑气泡宽度为 160dp，高度为 36dp
- **间距设置**：按键间距为 4dp，行间距为 6dp，键盘内边距水平方向为 4dp，垂直方向为 8dp
- **气泡指针**：与按键同高，确保完全覆盖按键


## 使用方法

### 调整键盘高度
1. 在键盘显示状态下，激活键盘调整模式
2. 拖动键盘上方的调整控件来改变键盘高度
3. 实时预览调整后的效果
4. 点击"确认"保存调整结果，或"取消"放弃更改

### 更改配色方案
1. 进入设置页面
2. 选择"配色方案"
3. 从多个预设方案中选择一个
4. 配色更改会立即在设置页面生效，键盘需要重启输入法后生效
