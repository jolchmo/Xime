# 表情

表情功能提供丰富的表情符号和自定义表情插件，让输入更加生动有趣。

## 功能特性

- **内置表情** - 包含笑脸、手势、动物、食物、自然、交通等多个分类的表情符号
- **表情插件** - 支持扩展表情插件，可添加颜文字、表情包、贴纸等
- **分类浏览** - 按类别浏览表情，便于快速查找
- **快速插入** - 点击表情即可插入到输入框中
- **插件扩展** - 可通过安装插件获取更多表情资源

## 使用方法

1. 点击键盘上的Logo按钮打开功能菜单
2. 选择“表情”进入表情界面
3. 浏览不同分类的表情符号
4. 点击表情即可插入到当前输入框
5. 如需更多表情，可安装对应的表情插件
