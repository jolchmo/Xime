# 输入方案

输入方案管理允许用户选择不同的输入方法，满足不同的输入习惯。

## 功能特性

- **多种方案** - 支持五笔86、五笔拼音混输等多种输入方案
- **方案切换** - 可随时切换到其他输入方案
- **当前状态** - 清晰显示当前激活的输入方案
- **即时生效** - 切换后重载配置即可应用新方案

## 使用方法

1. 点击键盘上的Logo按钮打开功能菜单
2. 选择“输入方案”
3. 在方案列表中选择想要使用的输入方案
4. 点击“重载配置”或“部署方案”使新方案生效
5. 重载完成后，新的输入方案即可使用

## 注意事项

- 切换输入方案后，必须执行重载配置才能生效
- 重载过程需要几秒钟，请耐心等待
- 不同输入方案可能需要不同的词库支持
