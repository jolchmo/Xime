# 部署方案

部署方案功能用于重新加载和应用输入方案配置，让修改后的设置生效。

## 功能特性

- **重载配置** - 重新加载当前的输入方案配置
- **应用更改** - 在切换输入方案或修改设置后需要重载配置
- **即时生效** - 重载完成后，新的配置会立即应用

## 使用方法

1. 点击键盘上的Logo按钮打开功能菜单
2. 选择“重载配置”或“部署方案”
3. 等待几秒钟，让系统完成配置重载
4. 重载完成后，新的输入方案或设置即可生效

## 注意事项

- 重载配置需要等待几秒钟，请耐心等待
- 切换输入方案后，必须执行重载配置才能生效
- 修改词库或表情插件后，也建议重载配置
