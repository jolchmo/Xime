# 剪贴板管理

剪贴板功能自动保存复制的内容，方便用户快速调用历史记录。

## 功能特性

- **自动保存** - 复制的文本会自动保存到剪贴板历史
- **分类查看** - 通过“剪贴板”和“快捷发送”两个标签分类管理
- **置顶功能** - 重要内容可置顶，避免被新内容覆盖
- **快捷发送** - 收藏常用内容到快捷发送，永久保存
- **批量管理** - 支持删除单条记录或清空全部历史

## 使用方法

1. 点击键盘上的Logo按钮打开功能菜单
2. 选择“剪贴板”进入剪贴板界面
3. 在“剪贴板”标签查看所有复制历史，在“快捷发送”标签查看收藏内容
4. 点击条目可直接插入到输入框
5. 点击星形图标可将内容添加到快捷发送
6. 点击图钉图标可置顶重要内容
7. 点击删除图标可移除单条记录
